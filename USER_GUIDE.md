# KidsClubPlans Assistant — User Guide

## Getting Started

**URL:** https://chat.kidsclubplans.app

No login required! The system creates a session automatically when you start chatting.

## Quick Examples

### Find Activities
```
"What indoor activities work for 8-year-olds?"
"Show me science experiments using household items"
"Physical games that don't need equipment"
```

### Plan a Schedule
```
"Plan a full day for 10-year-olds on Tuesday"
"Half-day schedule for a rainy day"
"4-hour outdoor day with breaks"
```

### Generate from Supplies
```
"I have paper plates, markers, and balloons. What can we do?"
```

### Blend Activities
```
"Combine dodgeball with storytelling"
"Blend art project with math lesson"
```

## Features

### 1. Chat Interface
- Type naturally — no special commands needed
- Responses stream in real-time
- Click cards to expand and see full details
- Previous conversation is remembered

### 2. Tools Panel ("Tools" button)

**Supplies Tab**
- Enter supplies you have on hand
- Select age group
- Get activity ideas instantly

**Blend Tab**
- Enter 2+ activity names
- Choose blend focus (physical, creative, educational, social)
- Get a novel combined activity

**Gaps Tab**
- Analyzes your activity database
- Shows what's missing (e.g., "only 2 winter activities")
- Suggests activities to fill gaps

### 3. Schedule Manager ("My Schedules" button)

**View Saved Schedules**
- Browse all your saved schedules
- Search by date or age group

**Schedule Actions**
- **Print** — Print-friendly view
- **Duplicate** — Copy schedule for another day
- **Export .ics** — Download for Google/Apple Calendar
- **Load in Chat** — Discuss schedule with AI
- **Supply Checklist** — See all supplies needed

### 4. Saving Activities

When the AI generates activities:
1. Click card to expand
2. Review details (description, supplies, instructions)
3. Click "Save to Database" to keep it
4. Or "Blend with Another" to remix it

## Tips & Tricks

### Better Results
- Be specific: "outdoor games for 10 kids" beats "games"
- Include constraints: "low prep", "no supplies", "rainy day"
- Ask for alternatives: "Give me 3 options"

### Managing Schedules
- Use templates: "Full Day" and "Half Day" quick buttons
- Duplicate successful schedules and tweak
- Export to calendar for reminders

### Activity Cards
- **Purple badge** = AI-generated activity
- **Blue badge** = Blended activity
- Click arrow to expand/collapse
- Instructions show as numbered steps

## Troubleshooting

### "Something went wrong"
- Refresh the page
- Start a new conversation (Clear button)
- Try rephrasing your question

### Activities not saving
- Check your internet connection
- Wait for "Saved!" confirmation
- Try again if it fails

### Schedule export not working
- Check popup blocker settings
- Try Print instead and save as PDF
